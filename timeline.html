Course 8 Sessions 60 min each
Session 1 60 min

What is AI?

Demystify AI — narrow vs general, AI they already use (Spotify, TikTok, Maps). Hands-on: prompt ChatGPT in different ways and see how the prompt controls the output.

lectureChatGPT demoprompts
Session 2 60 min

How LLMs Work

Tokens, prompts, completions, and the temperature dial. Challenge: write 3 prompts for the same question — plain, persona-based, and style-constrained — then compare.

tokenstemperatureprompt engineering
Session 3 60 min

What Are AI Agents?

Chatbot vs agent. The agent loop: Think → Act → Observe → Repeat. Live demo of instructor's agent on Telegram doing real searches and remembering things.

agent looptoolslive demo
Session 4 60 min

Setup Day

Students get their SSH keys, set up VS Code Remote SSH, connect to their cloud server, send their first message to their agent, and make their first edit.

SSH key setupVS Code Remotefirst message!
Session 5 60 min

How Your Agent Works

Workspace files = agent DNA. Edit SOUL.md to give it a pirate personality. Add rules in AGENTS.md. Experience the edit → restart → test loop.

SOUL.mdAGENTS.mdpersonality
Session 6 60 min

Tools & Capabilities

Web search, browser, and memory — what each does and when the agent uses it. Tool testing challenge: send specific messages and watch which tools fire.

web searchbrowsermemory
Session 7 60 min

Build Your Agent

Pick a problem you care about. Design personality, rules, and identity. Build in VS Code, test on Telegram, iterate. Homework: keep refining before Demo Day.

designbuilditerate

5-minute presentations per student — live demo on Telegram. Class votes on most creative agent. Recap everything learned and discuss what's next.
